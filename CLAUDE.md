# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

```bash
pnpm dev          # Start dev server (default port 4321)
pnpm build        # Build for production → dist/
pnpm preview      # Preview production build
```

> **Windows note:** Tailwind config must stay as `tailwind.config.cjs` with `module.exports` (not ESM). The `astro.config.mjs` explicitly points to it via `configFile: './tailwind.config.cjs'`. Changing to `.mjs` / `export default` breaks PostCSS on Windows.

## Architecture

### Data layer
- **`src/data/clinic.ts`** — single source of truth for all clinic NAP data (name, address, phone, hours, geo, social). Import this everywhere instead of hardcoding strings.
- **`src/content/services/*.json`** + **`src/content/doctors/*.json`** — Content Collections (Astro 5 glob loader). Schemas defined in `src/content.config.ts`.

### Pages
- `src/pages/index.astro` — homepage
- `src/pages/[about|services|doctors|contact].astro` — static pages
- `src/pages/services/[slug].astro` — dynamic route, one page per service JSON file

### Layout system
- `src/layouts/BaseLayout.astro` — wraps all pages; accepts `title`, `description`, `schemaMarkup` (array of JSON-LD objects). Adds `pt-16` body offset for the fixed navbar. Contains the IntersectionObserver script that drives `.appear` animations.
- `src/components/layout/Head.astro` — renders `<head>`: fonts preconnect, OG/Twitter meta, canonical, MedicalClinic JSON-LD + page-specific schemas.
- `src/components/layout/Navbar.astro` — fixed, full-width, 3-column grid (`grid-cols-3`): logo | nav | right-actions. Logo is the SVG from `public/lmc.svg` inlined via `fs.readFileSync` at build time (required because the SVG uses filters/masks that don't render in `<img>`).
- `src/components/layout/Footer.astro` — same inline-SVG logo technique.

### Styling
- Design tokens live in `tailwind.config.cjs` (`primary`, `ink`, `ink-secondary`, `surface`, `border`, `accent`, etc.).
- Reusable component classes (`.btn-primary`, `.btn-whatsapp`, `.card`, `.badge-*`, `.section-label`, `.section-title`, `.input`) are defined in `src/styles/global.css` under `@layer components`.
- Scroll-reveal: add class `appear` to any element — the BaseLayout IntersectionObserver toggles `in-view` on it. Staggered children: add `appear-group` to parent.

### SEO / AIO
- JSON-LD schemas are passed as the `schemaMarkup` prop to `BaseLayout` → rendered by `Head.astro`.
- `public/robots.txt` and `public/llms.txt` (AI search optimization) are static files.
- Sitemap auto-generated by `@astrojs/sitemap` on build.

### Build notes
- `@playform/compress` runs post-build to compress HTML/CSS/JS/SVG.
- `.npmrc` allowlists `esbuild` and `sharp` for pnpm build scripts (`onlyBuiltDependencies`).
